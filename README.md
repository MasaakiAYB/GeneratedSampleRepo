# GeneratedSampleRepo

## Frontend

Node.js 20 以上と npm を想定しています。

### セットアップ

```bash
cd frontend
npm install
```

### 開発サーバー起動

```bash
npm run dev
```

ブラウザで表示された URL（通常 `http://localhost:5173`）を開いて確認します。

### 品質確認（build）

```bash
npm run build
```
